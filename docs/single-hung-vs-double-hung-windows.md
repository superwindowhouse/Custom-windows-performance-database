# Single Hung vs Double Hung Window Engineering Comparison

## Overview
This document compares single hung and double hung window systems for building envelope applications. The primary engineering differentiator is the jamb liner and balance system, affecting ventilation control, maintenance accessibility, and lifecycle cost.

## Performance Parameters

| Parameter | Single Hung | Double Hung |
|-----------|-------------|-------------|
| Operating Sashes | 1 (Bottom) | 2 (Top & Bottom) |
| Air Infiltration Rate (AAMA Tested) | 0.1 – 0.3 cfm/ft² | 0.1 – 0.3 cfm/ft² |
| Air Infiltration Target (High-Performance) | <0.15 cfm/ft² at 1.57 psf (75 Pa) | <0.15 cfm/ft² at 1.57 psf (75 Pa) |
| Design Pressure (DP) Rating Specification | DP 50+ | DP 50+ |
| Typical Unit Cost (Installed) | $150 – $350 | $250 – $600 |
| Annual Maintenance Cost (Estimated) | ~$15 | ~$10 |

## Material Configuration

| Component | Specification |
|-----------|---------------|
| Frame Material | PVC (Polyvinyl Chloride) / Vinyl |
| Frame U-Value | 0.20 – 0.30 |
| Weatherstripping | Compression seals / Dual-fin bulb seals |
| Balance System | Block-and-tackle type |
| Sash Operation Force (Tilt-latch) | <30 lbf (per AAMA standards) |

## Structural Characteristics

- Single hung: Fixed upper sash, movable bottom sash → fewer balance mechanisms and seals
- Double hung: Both upper and lower sashes operable → tilt-latch mechanism for inward rotation
- Double hung: Top sash opening 3–4 inches enables natural convection ventilation
- Double hung: Top sash provides secondary egress path for code compliance
- Single hung: Static top sash allows permanently sealed jamb liner

## Thermal / Energy / Air Performance

| Parameter | Value |
|-----------|-------|
| Frame Thermal Resistance (U-Value) | 0.20 – 0.30 |
| Air Leakage (Both Types, AAMA) | 0.1 – 0.3 cfm/ft² |
| Air Leakage (Optimized Double Hung) | <0.15 cfm/ft² at 1.57 psf (75 Pa) |

## Applicable Standards

- AAMA/WDMA/CSA 101/I.S.2/A440
- NFRC
- ENERGY STAR
- AAMA (tilt-latch operating force)

## Engineering Notes

- Air infiltration performance is not inherently determined by window type (single vs double hung). Jamb liner and compression weatherstripping quality are the decisive factors.
- A 50-unit apartment building cost analysis: single hung upfront savings ($5,000–$10,000) may be offset within 3–5 years by exterior cleaning costs for multi-story applications.
- Vinyl frames provide corrosion resistance for coastal environments and can be extruded into multi-chamber profiles for balance and weatherstripping chases.
- For high-rise applications, tilt-latch operating force must be <30 lbf per AAMA standards.
