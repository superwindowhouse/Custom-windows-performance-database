## Overview

This document covers vertical sliding vinyl hung windows, available in single hung and double hung configurations, for multi-family renovations, historic building retrofits, and new construction applications.

## Material Configuration

- Frame material: Vinyl formulation (glass-reinforced compounds for sash corners)
- Glazing options: Dual-pane, low-E/argon, or triple-pane configurations
- Frame depth: 3-1/4" to 4-9/16" to match historic wood window jambs
- Custom slit dimensions for full-frame replacements: fits existing 1-1/2" to 2" thick walls
- Color: Vinyl formulations replicating historical paint colors with UV stability

## Structural Characteristics

- Balance system types:
  - Spiral balances: 300–400 lb capacity for 6-foot tall windows
  - Block-and-tackle systems: For historic profiles requiring concealment
  - Constant force coil springs: For tilt-in cleaning functionality
  - Friction shoe assemblies: For wind load resistance in high-rise installations
- Sash corner structural strength provided by glass-reinforced vinyl compounds
- Nailing fin fasteners: 6" on center perimeter, 4" on corners, using corrosion-resistant screws
- Rough opening tolerance: +1/2" in width, +1/4" in height above frame dimensions

## Engineering Notes

- Lifecycle cost advantage exceeds $800 per opening over 30 years.
- Installation labor reduced by 30% compared to casement or awning alternatives.
- Flashing sequence: bottom first, sides overlapping bottom, top overlapping sides.
- Shim placement: leveling shims at sill; never at jambs.
- Four-person crews average 8–12 units per day.
- Simulated divided lights with true internal grilles between glass panes.
