# Vinyl Casement Window Engineering Performance Specification

## Overview

Vinyl casement windows are crank-operated, side-hinged fenestration units that compress the sash against a multi-point locking system.

## Performance Parameters

| Parameter | Specification |
|-----------|---------------|
| Air Infiltration Rate | ≤ 0.05 cfm/ft² (AAMA/WDMA/CSA 101/I.S.2/A440) |
| U-Factor (Commercial Grade) | 0.26 – 0.30 |
| U-Factor (Center-of-Glass) | ≥ 0.18 |
| SHGC (Selective Low-E) | 0.22 – 0.28 |
| Condensation Resistance (CR) | 65 – 75 |
| Structural DP Rating (Commercial Grade) | 50 – 65 |
| Egress Clear Opening | ≥ 5.7 ft² |
| Egress Clear Width | ≥ 20 in |
| Egress Clear Height | ≥ 24 in |
| Egress Sill Height (from finished floor) | ≤ 44 in |

## Material Configuration

| Component | Specification |
|-----------|---------------|
| Frame Extrusion | Multi-chambered vinyl (5–7 chambers) |
| Corner Joints | Fusion-welded |
| Weatherstripping | Bulb-type EPDM or silicone gaskets |
| Reinforcement | Internal aluminum or galvanized steel (for DP 50–65) |
| Glass | 3/4 in insulated glass |

## Structural Characteristics

- Side-hinged mechanism with crank operation
- Three-sided compression seal when closed
- Nearly 100% unobstructed clear opening area
- Multi-point locking hardware
- Integral weatherstripping on sash and frame (two continuous seal lines)

## Thermal / Acoustic / Energy Performance

| Metric | Standard Builder Grade | Commercial Grade |
|--------|-----------------------|-----------------|
| U-Factor | 0.35 – 0.40 | 0.26 – 0.30 |
| SHGC | 0.30 – 0.40 | 0.22 – 0.28 |
| Air Infiltration (cfm/ft²) | 0.30 | 0.05 – 0.10 |
| DP Rating | 35 – 40 | 50 – 65 |
| Condensation Resistance | 45 – 55 | 65 – 75 |

## Applicable Standards

- AAMA/WDMA/CSA 101/I.S.2/A440
- ASTM E1886/E1996 (missile impact resistance)
- NFRC 100 (U-Factor)
- 2024 International Residential Code (IRC) (egress requirements)
- 2024 IECC (U-factor ≤ 0.30)

## Engineering Notes

- Egress compliance requires verifying actual clear opening dimensions, not nominal frame size.
- Crank handle must not intrude into the clear opening during measurement.
- For EIFS-clad buildings, attach window to structural framing before EIFS application; do not fasten through EIFS.
- For multi-story installations, each window must be leveled independently.
