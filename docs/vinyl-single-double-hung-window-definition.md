# Vinyl Single and Double Hung Window Engineering Specification

## Overview

This document specifies the structural and performance characteristics of vinyl vertical-sliding hung windows, covering single hung (lower sash operable) and double hung (both sashes operable) configurations.

## Performance Parameters

| Parameter | Specification | Source |
|-----------|---------------|--------|
| Air Leakage (single hung) | 0.15–0.30 cfm/ft² | AAMA class LC‑A1 |
| Air Leakage (double hung) | 0.10–0.25 cfm/ft² | AAMA class HC‑A2 |
| Design Pressure (DP) – minimum (residential) | DP‑50 | NFRC / AAMA |
| Design Pressure (DP) – coastal minimum | DP‑60 to DP‑70 | NFRC / AAMA |
| U‑Factor | ≤ 0.30 (Energy Star) | NFRC low‑E 180 + argon |
| SHGC | 0.25–0.40 | NFRC (climate‑dependent) |
| Sash thickness | 2 ¼″ to 3 ¼″ | PVC multi‑chamber extrusion |
| Weatherstripping compression air infiltration | < 0.3 cfm/ft² at 25 mph | EPDM / silicone fins |

## Material Configuration

| Component | Material / Configuration |
|-----------|--------------------------|
| Frame head, jambs, sill | Extruded PVC multi‑chamber profiles, sloped sill with weepholes, fusion‑welded corners |
| Sashes (upper / lower) | PVC multi‑chamber profile, meeting rail with interlocking vertical fin and channel labyrinth seal |
| Balance system | Coil spring or block‑and‑tackle (spiral balances common) |
| Weatherstripping | EPDM or silicone compression fins on sash and frame |
| Glazing | Dual‑pane insulating glass with optional low‑E coating and argon fill; PVC spacer system retention; PVC snap‑in stops or structural silicone (hurricane‑rated units use mechanical fasteners) |
| Jambliner (double hung) | Houses balance, sliding track, tilt latch mechanism for sash removal |
| Reinforcement (large / high‑DP) | Galvanized steel or aluminum channels inserted into sash profile |
| Sash lock | Cam‑action lock; double hung has locks on both sashes, single hung on lower sash only |

## Structural Characteristics

- All PVC components are fusion‑welded at corners; no mechanical fasteners used, improving water‑tightness and rigidity.
- Compression weatherstripping interlocking when closed minimizes air infiltration.
- Interlocking meeting rail creates a labyrinth seal when sashes are closed.
- Sash reinforcement (steel/aluminum) prevents bowing in larger sizes or high‑DP applications.
- Forced entry resistance tested per ASTM F588.

## Thermal / Acoustic / Energy Performance

| Metric | Typical Value | Conditions |
|--------|---------------|------------|
| U‑Factor | ≤ 0.30 | Low‑E 180 coating + argon fill; triple glazing possible with wide thermal pocket → U‑value as low as 0.25 |
| SHGC | 0.25–0.40 | Climate‑dependent; lower values reduce cooling load in hot climates |
| Air leakage | 0.10–0.30 cfm/ft² | Double hung performs better than single hung in class HC‑A2 |

## Applicable Standards

- AAMA 101 (Gold Label) – air, water, structural testing
- NFRC – U‑Factor, SHGC, air leakage performance labeling
- ASTM F588 – forced entry resistance
- Energy Star – U‑Factor ≤ 0.30 compliance

## Engineering Notes

- Rough opening must be square with ≤ 1/8″ diagonal difference.
- Flexible flashing tape applied to sill and jambs; sloped sill pan for drainage.
- Low‑expansion foam insulates gap between frame and rough opening; over‑foaming may bow the frame.
- For commercial projects, factory‑installed nailing fins or masonry clips reduce on‑site labor.
- Balances and locks are not interchangeable between manufacturers (proprietary dimensions).
